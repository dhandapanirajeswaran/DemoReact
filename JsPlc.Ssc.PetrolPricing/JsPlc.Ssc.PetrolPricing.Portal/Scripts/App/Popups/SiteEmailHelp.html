Site Email Help

Using the Editor

To edit a section of the Site email, simply click on the area you wish to edit.

For complex sections (such as the Email Body itself) you might have to click a second time to select a particular item to edit.

Learning about the Site Email editor
Opens or closes the Help window.

Email Templates

The appearance of an Email is controlled by an Email Template. The template consists of static text, styles and tokens (which are placeholders for site-specific values).

Note:You cannot edit or delete the "Default Email Template".

Template controls
Deletes the currently selected email template.
Note: You cannot delete the "Default Email Template"
Makes a copy or clone of the currently selected email template, prompting you to enter a name for the template.

Importance

The Importance represents the styling of individual items of email text. The important style will affect the font colour and weight.

To change the importance/appearance of an email item, simply click it and change its importance in the dropdown menu.

Currently there are 4 difference importance types

  1. Normal — Normal text and weight.
  2. Information — Low importance. Text will appear in green.
  3. Warning — Medium importance. Text will appear in orange.
  4. Urgent! — This is the high importance. Text will appear in red.

{ Tokens }

Tokens represent placeholders which are replaced before the email is sent.

You can either manually type the token into the editor input box or use the dropdown menu and press Add

Inserting tokens using the menu.
Inserts the selected Token at the end of the item you are currently editing.

The currently supported tokens are:-

{SiteName}
Name of the current site
{DayMonthYear}
The effective date of the change (e.g. 30/05/2017)
{UnleadedPrice}
The price change for Unleaded
{SuperPrice}
The price change for Super-Unleaded
{DieselPrice}
The price change for Diesel

The main editor controls

Note: The control panel will only show controls which are relevant to the item you're editing.

The controls provide basic editing, re-ordering, deleting and adding of small set of HTML elements.

Changing the order of items in the Email Template
Moves the currently selected item up or down.
Adding new items
  1. Select the type of item you wish to add using the dropdown menu
  2. Click the Add button.

The new item will appear after the currently selected one.

Tip: you can always change the order afterwards using the Up and Down buttons.
Saving the changes / Closing the editor
Saves all the changes to the Email Template.
Closes the editing window (also prompts to save any unsaved changes).

Tables, Rows, Columns and Cells

Table represents grid-based tabular information. It's structure consists of a number cells arranged in horizontal Rows and vertical Columns

The following controls are only available when editing a table cell.

Table Rows, Columns and Cells
Move the entire table Column either left or right.
Adds a new Column after the current cell
Deletes the currently selected Column from the table.
