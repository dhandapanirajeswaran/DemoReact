User Management Page

From the User Management page you can manage users of the Petrol Pricing tool.

Adding a new user

You can add a new using by clicking on the button.

This will ask for a First Name, Last Name and Email for the new user.

Currently you cannot edit an existing user's details so you will need to delete and then re-added them with the correct details.

Delete an existing User

To delete an existing User find their user in the grid and click on the button

You cannot delete yourself from the Petrol Pricing tool.
