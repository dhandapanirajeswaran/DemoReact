How can I...

This section explains how to complete some commonly asked about tasks within the Petrol Pricing tool.

How to Manage Price Freezes

You can manage the Price Freezes on the Price Freeze Event page.

Uploading Files

You view the recently uploaded files on the File Uploads page. From there you can Upload a new file.

Tip: You can find further help on File Uploads on the different types of uploads.

Pricing Sainsburys Fuel Sites

Within the main pricing page you can view, search and manage Sainsburys fuel site prices for Unleaded, Diesel and Super-Unleaded.

Bulk Importing of Site Emails

Using the Site Emails page you can view the Sainsburys sites, their StoreNo, CatNo, PfsNo and Email Addresses.

  1. Firstly, to export all the currently values
  2. Make any changes to the Excel spreadsheet (such as changing CatNo or PfsNo)
  3. Use the button to view the Upload Site Emails

How can I manage users

You can manage user on the Users page. On this management page you can:

Users to the Petrol Pricing tool can use the normal Windows username (email) and their password.

  1. View existing users
  2. Add new users
  3. Delete existing users
  4. See when a user was added
  5. See when the user last logged in

No passwords are stored within the Petrol Pricing tool - they are all stored within Microsoft's single sign-on
