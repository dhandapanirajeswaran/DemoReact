Types of file uploads

There are the 4 types of file upload.

  1. Daily Price Data
  2. Quarterly Site Data
  3. Latest JS Price Data
  4. Latest Competitors Price Data

You may also be interested in the Email Management file upload

Daily Price Data File

The Daily Price Datafile (also known as the Catalist file) is normally uploaded once per day. It contains prices for both Sainsburys and our Competitor sites.

The file itself contains all three fuel grades (Unleaded, Diesel and Super-Unleaded)

Quarterly Site Data File

This file contains all the Site information such as Name, Town, Distance, Drive Time, Brand, Postcode and Ownership.

This is normally uploaded 4 times per year (quarterly) and also during System Setup

Latest JS Price Data File

This file is uploaded for a particular day to import Sainsburys price information for that day.

Latest Competitors Price Data file

This (as the name suggests) contains all the Price data for our competitors (such as Asda, Tesco and the other independant sites).

File Templates

For the main 3 File upload types (Quarterly Site Data, Latest Js Price Data and Latest Competitor Price Data) files there is a button. This file should be downloaded, populated with the appropriate data and then uploaded into the system.

Date Picker and Historical data

During normal usage the date picker will only appear when uploading a Daily Price Data file.

You enable a date picker for all four Upload Types using enabling the option on the System Setting page. This can be done to allow historical data to be uploaded in the system during initial setup (or testing).
