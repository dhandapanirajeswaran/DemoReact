Site Email Settings

This controls if the Testing Email Address or the LIVE Site Email Addresses will be used when sending out Price Changes to the Sainsburuys fuel sites.

Common System Settings

The Common System Settings section has the following settings

Nearby Grocer Drive Time
The number of minutes used to find Nearby Grocers
Price Change Variance
The +/- threshold used to detect small price changes. If the price change is within or equal to the Price Change Variance then the price will be snapped back to Today price.
Decimal Rounding
This menu specifics if the number after the decimal point in a price should be replaced and with what the number digit should be.
Super-Unleaded Markup
This value is the number of pence that will be added to the Unleaded Price to calculate the Super-Unleaded price. Normally this is 5.0 which means Super-Unleaded is +5.0p more expensive than Unleaded.
Show Unload Date Picker
This enables/disables whether the Date Picker should appear for all the 4 File Upload types.
This is useful when uploading historical data during System Setup
Maximum Competitor Drive Time
This is the maximum number of minutes (drive time range) when searching for the cheapest Competitor price.

Price Overrides limits and warnings

These numbers defined the Absolute Min/Max Price limits and the Warning Indicator limits for each of the fuel types.

The Absolute Min/Max prevent the user from entering an Override price below the Min or higher than the Max limit.

The Warning Indicator limits are prompts to warning, but not prevent, the user from making large price changes. (E.g. a price change less than -5.0p or higher than +5.0p)

Drive Time Markup

For each of the 3 Fuel Types (Unleaded, Diesel and Super-Unleaded) you can define the Drive Time markup based on the drive time to a Competitor site.

You can manage the Drive Time Mark.

Removing
To remove an existing Drive Time Markup click on the button.
Editing
To edit an existing Drive Time Markup click on the row itself and then change the Drive Time and Markup values. Once finished press the button.
Adding
To add a new Drive Time Markup enter values into the Drive Time and Markup fields and then click the button.

Brands & Grocers

From the Brands & Grocers page you can manage the Excluded Brands and Grocers

A Exclude Brand will prevent that Brand from being used when searching for the cheapest Competitor.

A Grocer is our main (highest grocing) Competitors and can include ourselves. (e.g. ASDA, Morrisons,Tesco and Sainsburys)

Email Schedule

The Email Schedule page allows you to change the time of the scheduled email report, view the recent Schedule Event Log entries, clear down the log or disable the scheduled email.

The Scheduled Item currently will send out the Daily Price Email report to the specified recipient in the Email Address.

Once you Edit the Schedule Item and press the button the Scheduled For column will display Today at HH:MM. This means the email has not been sent out today.

Clicking on the will send out any pending (Scheduled For) emails for today. The Scheduled For column will then be updated to show Tomorrow at HH:MM.

Price Freeze Events

On the Price Freeze Events page you can manage a number of Price Freeze Events.

Inactive Events
These are Price Freeze Events which will not be used for their date range.
Expired Events
These are Price Freeze Events whose end date is in the past.
Future Events
These are Price Freeze Events whose start date is in the future.
