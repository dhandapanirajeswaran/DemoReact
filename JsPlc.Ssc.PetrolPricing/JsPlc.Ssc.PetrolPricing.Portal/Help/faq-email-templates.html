Using an email template

On the Pricing page hover over a site name and click the button to open the email template modal.

Choose an existing template using the Template menu or press the clone button to duplicate the existing template.

Review the banner at the top showing whether the Test Email Addresses or the Live email addresses will be used then sending the emails.

Tip: You can switch between the Test and Live email addresses on the Settings page

Editing email templates

Choose an existing email template (not the default template) and then click either the Subject Line or the Email body region to begin editing the email template.

For the Email Body you can also click individual components of the email body (such as a paragraph, table cell or text) to begin editing it. Once editing you can re-arrange the vertical order of email components, change styles or insert Tokens

You can change the appearance of email templates in the following ways:

  1. Add, remove or edit text
  2. Add, remove or edit headings
  3. Add Tokens which will be replaced with actual values when sending emails
  4. Re-arrange the vertical order of email components
  5. Add or remove Table Cells
  6. Add Bullet points, Paragraphs, Table Cell or Heading

You can not delete or edit the default email template.

You can copy/clone any existing email template (include the default one) and edit as you wish.

Email Tokens

These are placeholder for actual values will be replaced when the email is sent out. Currently there are 5 tokens

  1. Site Name
  2. Current Date
  3. Unleaded Price
  4. Diesel Price
  5. Super-Unleaded Price
