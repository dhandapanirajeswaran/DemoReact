Clearing down the data

Certain users (with relevant permissions) can clear down the data within the system itself. This is done on Diagnostics page.

This feature is intended for testing and the initial setup of the Petrol Pricing tool and not for the day-to-day usage since any existing data is erased and cannot be recovered.

Loading Seed Data

After the system has had its data cleared down you will need to upload the seed data. This "seed" data is the Quarterly Site Data file along with any Historic Daily Price Data (aka Catalist) files.

You can upload these files on the File upload page
